GitHub Repositories & Portfolio

Here are my side projects as part of my education in software development. Some of these projects have source code available on GitHub; to access it, you can check out either my GitHub profile, or click on one of the projects below to go directly to that project’s corresponding repository.
You should get familiar with how to navigate through these projects before accessing them, as they are stored in individual version control memory groups called repositories. Each link takes you to the root directory of the project, which holds all the files of the project in a tree-like structure.

Gregory Desrosiers' Animated Balls – Lite Version

Built: August 2013

One side project out of plain silliness with Java's AWT framework. Includes the source code and an executable file as a JAR file. "Help" and "About" features are encoded in the program for you. However, the source code itself is not commented, and so if you decide to read it, do so at your own risk.
